# ITviec Cypress 13 Project

Dự án này sử dụng Cypress phiên bản 13 để kiểm thử tự động cho ITviec.

## Cấu trúc thư mục
- `cypress/e2e`: Chứa các test case end-to-end
- `cypress/fixtures`: Chứa dữ liệu mẫu cho test
- `cypress/support`: Chứa các file hỗ trợ, custom commands
- `cypress.config.js`: File cấu hình Cypress

## Hướng dẫn cài đặt
1. Cài đặt Node.js >= 18
2. Chạy lệnh `npm install`
3. Chạy test với lệnh `npx cypress open` hoặc `npx cypress run`
